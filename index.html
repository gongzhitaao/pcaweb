Content Control

This site allows for the disabling advertisements. Disabling ads will remove the ads from the left hand side. The choices below will set a cookie on your machine that will let the site know when to disable content. The cookie only keeps track of what content to disable and personal information is not stored. The site will still run with cookies disabled, but you will not be able to disable any content.
